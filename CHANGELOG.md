# Changelog

All notable changes will be documented here.

## [1.0.0] - 2026-07-24

- Prepare the 1.0.0 release with application branding and UI consistency improvements.
- Prevent knowledge-base rebuilds when no supported source documents are available.
- Keep the Knowledge Base dialog open during uploads and show per-file status.
- Add persisted default Agent Profiles and reliable in-app deletion confirmation.
- Add individual document deletion, clear-all, and new-index workflows.
- Fix overlap progression that could create thousands of duplicate micro-chunks.
- Increase default chunks to 1,600 characters with 200-character overlap.
- Prepare the project for its first public release.
- Default browser mode to localhost-only access.
- Isolate browser streaming events by session.
- Add API allowlisting and upload validation.
- Preserve original attachment filenames.
- Add retrieval distance filtering, tests, CI, and community files.
- Add persistent Agent Profiles with per-chat selection and JSON import/export.
- Add an About view with runtime, model, version, repository, and license details.

## [0.1.0] - 2026-07-23

Initial public preview.
